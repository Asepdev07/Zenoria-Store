What is ZenoriaMC?

ZenoriaMC adalah server minecraft yg berfokus pada sistem economy yang menghadirkan pengalaman bermain yang menarik untuk Java dan Bedrock.

Sistem Economy (Zen Economy)

ZenoriaMC menghadirkan sistem economy yg mendalam (untuk kedepannya). Kami juga akan mencampur sistem seperti Claim Block Shop dan Cosmetic Shop. Jadi tunggu apalagi, yuk dukung server ini dengan membeli rank!
